# История версий

Формат вдохновлён [Keep a Changelog](https://keepachangelog.com/ru/);
версии — [SemVer](https://semver.org/lang/ru/). Полные диффы — в
[релизах GitHub](https://github.com/xvin84/Adalight/releases).

## [0.14.0] — 2026-07-19

- Окно «Что нового» после обновления: список изменений (в т.ч. через несколько
  версий) на языке интерфейса, с переключателем языка прямо в окне.
- Плагины из каталога ставятся без перезапуска; установленные показываются
  серыми с пометкой «Установлено».
- Языки видны в списке «Установленные» менеджера — «Сделать языком интерфейса»;
  английский идёт через тот же контракт `create_locale()`, что и плагины-локали.
- Языки сообщества ставятся из каталога и появляются в списке сразу.
- Исправлена вёрстка настроек уведомлений (обрезанная подсказка, наложение
  пикера позиции при широком шрифте).

## [0.13.0] — 2026-07-19

- Локализация интерфейса: русский и английский (Система → Язык), смена языка
  применяется после перезапуска.
- Языки — это плагины: свой перевод добавляется файлом с `create_locale()`
  в папке плагинов; шаблон — examples/locales/en.py.
- Английский встроен и доступен сразу, без установки.

## [0.12.0] — 2026-07-18

- Менеджер плагинов отдельным окном: «Установленные» (вкл/выкл, настройки,
  удаление) и «Каталог» (поиск, установка). Вкладка в сайдбаре — сводка.
- Плагины объявляют `settings_schema` — менеджер строит форму настроек сам,
  без кода GUI (первый шаг к «всё есть плагин»).
- Вспышка уведомлений «бульк»: капля с расходящейся по ленте волной (стиль
  на выбор — «бульк» или «пятно»).
- Позиция вспышки задаётся перетаскиванием по краю экрана (только периметр).
- Подсказка в настройках: ловятся только системные уведомления Windows.
- Кнопки «Сообщить об ошибке» / «Предложить идею» в «Системе» открывают
  преднаполненный GitHub-issue с диагностикой.

## [0.11.0] — 2026-07-17

- Позиция вспышки уведомлений задаётся перетаскиванием пятна по схеме экрана
  (с мгновенной пробной вспышкой на ленте при отпускании).
- Каталог плагинов: официальные и от сообщества, установка в один клик.
- Режим «любые приложения»: вспышка цветом иконки приславшего уведомление.

## [0.10.0] — 2026-07-17

- Документация плагинов (docs/PLUGINS.md) и рабочий шаблон
  (examples/plugins/break_reminder.py); кнопка «Открыть папку плагинов».
- Эффекты лампы: Комета, Северное сияние, Звёздное небо.
- Эффекты цветомузыки: Волны от баса, Вспышки на битах.

## [0.9.x] — 2026-07-17

- Система плагинов: свои .py-файлы с `create_plugin()`, изоляция ошибок,
  настройки в общем конфиге.
- Встроенный плагин «Вспышки уведомлений»: Telegram — голубая, Discord —
  фиолетовая, поверх любого режима.
- WLED-UDP транспорт (beta): лента на ESP по Wi-Fi, DRGB/DNRGB.
- Тихое автообновление при запуске (по галочке).
- Быстрые эффекты лампы из меню трея.

## [0.8.x] — 2026-07-17

- Вертикальный сайдбар с SVG-иконками, статус-карточка, тосты,
  дружелюбные ошибки, мастер первого запуска, микроанимации.
- Кликабельный предпросмотр: клик по диоду вспыхивает им на ленте.
- Эффект «Камин» с настройками (высота, интенсивность, искры).
- Уведомления в трее; кнопка сохранения текущего профиля с индикатором.
- Исправлены гонка автообновления и утечка окружения PyInstaller
  («Python312.dll в Temp»), одиночный экземпляр, автозапуск Windows.

## [0.7.0] — 2026-07-17

- Встроенные пресеты 🎬 Кино / 🎮 Игра / 💼 Работа поверх настроек железа.
- Дизайн-система QSS: тёмная и светлая темы из одних токенов.

## [0.6.0] — 2026-07-16

- Профили настроек с переключением из окна и трея; импорт/экспорт в JSON.
- Баланс белого (множители R/G/B); окно запоминает геометрию и вкладку.

## [0.5.x] — 2026-07-16

- Интерфейс на вкладках; живой предпросмотр экрана с зонами сбора цвета.
- Тёмная/светлая/системная тема; автообновление из GitHub Releases;
  установщик Windows (Inno Setup); одиночный экземпляр.
- Гибкий градиент (2–8 точек); статичная радуга; чувствительность
  цветомузыки; иконка приложения.

## [0.4.0] — 2026-07-16

- Режимы «Лампа» (цвет/градиент/радуга/дыхание) и «Цветомузыка»
  (спектр, пульс от баса) поверх loopback-звука.
- Ночной режим; цветовая температура; порог теней; проверка обновлений;
  Linux-бинарник в релизах.

## [0.3.x] — 2026-07-16

- Живое применение настроек без ресета платы; автоприменение через 5 с.
- Расписание яркости (интервалы через полночь); адаптивная яркость.
- Автозапуск при входе в систему; бэкенд bettercam (обход бага dxcam).

## [0.2.0] — 2026-07-16

- Переписан в пакет с GUI (PySide6): порт, диоды по сторонам, угол,
  направление, порядок каналов; трей; предпросмотр.
- Захват Windows (DXGI) и Wayland; CI и сборка релизов по тегу.
